# ⚽ 오마카세미루

블랙 프라이데이 축구 유니폼 세일 총정리 사이트

## 📌 프로젝트 소개

국내외 쇼핑몰의 축구 유니폼 가격을 한눈에 비교할 수 있는 웹사이트입니다.
프리미어리그, 라리가, 세리에 A 등 유럽 주요 리그 팀들의 유니폼 최저가를 실시간으로 확인할 수 있습니다.

## ✨ 주요 기능

### 사용자 페이지 (index.html)
- 🔍 다양한 필터 옵션 (팀, 키트 유형, 시즌, 버전)
- 💰 최저가 자동 비교 (원화 환산)
- 🏷️ 최대 할인율 표시
- 📊 다양한 정렬 옵션 (기본순, 최신순, 가격순, 할인순)
- 📱 반응형 디자인

### 관리자 페이지 (admin.html)
- 🔒 비밀번호 보호
- ➕ 제품 추가/삭제/편집
- 🏪 판매처 관리
- 💱 실시간 환율 정보
- 👁️ 제품 노출 on/off 토글

## 🛍️ 제휴 쇼핑몰

- 아디다스 코리아 (공식 홈페이지)

## 🚀 배포

GitHub Pages를 통해 배포됩니다.

- **main 브랜치**: 개발 및 소스코드 관리
- **gh-pages 브랜치**: 실제 배포용

## 🔐 관리자 접근

관리자 페이지는 비밀번호로 보호됩니다.
기본 비밀번호: `casemiro7`

⚠️ 배포 전 admin.html 파일에서 비밀번호를 변경하세요!

## 📁 프로젝트 구조

```
omacasemiro/
├── index.html          # 메인 페이지
├── admin.html          # 관리자 페이지
├── product.html        # 제품 상세 페이지
├── js/
│   ├── data.js        # 제품 데이터
│   ├── main.js        # 메인 페이지 스크립트
│   ├── admin.js       # 관리자 페이지 스크립트
│   └── product.js     # 제품 상세 스크립트
├── clean_sellers.js   # 판매처 정리 스크립트
└── update_sellers.js  # 판매처 업데이트 스크립트
```

## 💻 로컬 개발

1. 저장소 클론
```bash
git clone https://github.com/YOUR_USERNAME/omacasemiro.git
cd omacasemiro
```

2. 브라우저에서 `index.html` 열기
