## 📋 판매처 추가 가이드라인

### 1. 정품 확인 필수 체크리스트
- [ ] 브랜드/구단 공식 스토어인가?
- [ ] 공식 수입 대리점/유통사인가?
- [ ] 정품 보증서를 제공하는가?
- [ ] AS 정책이 명확한가?
- [ ] 공식 홈페이지에 리테일러로 등록되어 있는가?

### 2. 플랫폼 판매 시 추가 확인
- [ ] '브랜드 직영' 또는 '공식 스토어' 마크가 있는가?
- [ ] 판매자가 공식 브랜드 계정인가?
- [ ] 상품 설명에 정품 인증 내용이 있는가?

### 3. 의심스러운 경우
- 너무 저렴한 가격 (정가의 50% 이하)
- 새 시즌 제품인데 재고가 과도하게 많음
- 판매자 정보가 불명확함
- 리뷰에 가품 언급이 있음
